# bio_project
# FASTA Stats Tool

CLI tool to compute basic statistics from FASTA files.

## Features
- Sequence length
- GC content
- Multi-file support
- CSV output

## Installation

```bash
git clone https://github.com/abhisheksny/bio_project.git
cd bio_project
python3 -m venv venv
source venv/bin/activate
pip install biopython
